# First Codex Starter Prompt

Use this prompt to create the smallest useful frontend version of the archived
AI Footprint Lab idea: one simple homepage that explains the project clearly.

```text
Context:
I am a beginner frontend developer. This repo has an archived project idea called AI Footprint Lab in docs/archive/ai-footprint-lab/project-context.md.

Goal:
Help me start with the smallest useful frontend version: a simple homepage for AI Footprint Lab.

Constraints:
Use Next.js, React, TypeScript, and Tailwind CSS. Frontend only. No backend, no APIs, no database. Keep the code beginner-readable.

Acceptance criteria:
- Scaffold the app if needed
- Create one homepage
- Include a hero title, short mission statement, and 3 preview cards:
  1. Hidden infrastructure of AI
  2. Benefits and trade-offs
  3. Future solutions
- Keep styling simple and clean
- Explain every file changed in beginner language
- Suggest one commit message

Ask:
Make the smallest safe first implementation.
```
