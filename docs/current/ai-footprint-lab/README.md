# AI Footprint Lab Current Docs

These files are the active planning, curriculum, and product-context docs for
AI Footprint Lab.

AI Footprint Lab is the current product direction for this repo. It is a
frontend-only educational website about the environmental and economic
trade-offs of artificial intelligence.

## Current Files

- `project-context.md`: active product vision and constraints.
- `nine-week-curriculum.md`: week-by-week build curriculum.
- `first-codex-starter-prompt.md`: starter homepage prompt.
- `day-1-research-before-code.md`: research-first lesson plan.
- `day-1-ai-research-lesson.md`: day-one AI research lesson.
- `week-2-source-backed-statistics-spec.md`: quantified statistics section spec.
- `brain-vs-ai-slider-ux-spec.md`: interactive Brain vs AI slider spec.
